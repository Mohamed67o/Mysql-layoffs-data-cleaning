# Layoffs Data Analysis Project

## Project Overview

This project analyzes a global layoffs dataset using MySQL.

The project focuses on:

- Data Cleaning
- Data Standardization
- Handling Missing Values
- Removing Duplicates
- Exploratory Data Analysis (EDA)
- Business Insights

---

## Skills Demonstrated

- SQL
- Data Cleaning
- Aggregate Functions
- CTEs
- Window Functions
- GROUP BY
- HAVING
- ORDER BY

---

## Project Steps

### Data Cleaning

- Removed duplicate records
- Standardized company and country names
- Handled null values
- Converted date fields

### Exploratory Data Analysis

- Top companies affected by layoffs
- Top industries affected
- Top countries affected
- Yearly layoffs trends
- Monthly rolling layoffs trends

## Key Insights

- Certain industries experienced significantly higher layoffs than others.
- Layoffs increased sharply during specific periods.
- A small number of companies accounted for a large percentage of layoffs.
